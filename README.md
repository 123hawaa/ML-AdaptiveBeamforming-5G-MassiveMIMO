# ML-AdaptiveBeamforming-5G-MassiveMIMO
Intelligent Machine Learning Model for Adaptive Beamforming in 5G Massive MIMO Millimeter Wave Systems
This project evaluates and enhances the performance of an adaptive beamforming algorithm in 5G millimeter-wave multicellular networks with massive MIMO at all active base stations. It integrates Non-Orthogonal Multiple Access (NOMA) to improve spectrum utilization and employs a machine learning (ML)-based selection model using k-nearest neighbors (k-NN) to simplify beamforming decisions.

The adaptive beamforming dynamically adjusts beam directions according to user distribution, improving Spectral Efficiency (SE) and Energy Efficiency (EE). Monte Carlo simulations show that the proposed NOMA-based algorithm achieves ~57.8 bps/Hz SE and ~7.8 Mbits/J EE, compared to the baseline’s 36 bps/Hz SE and 5 Mbits/J EE, while reducing the blocking probability to 0.5.

The ML component reduces computational complexity by selecting the most suitable predefined beam configuration instead of calculating beams in real-time, maintaining an accuracy of 98.7%. Selective activation of antenna subarrays also minimizes power consumption, making the approach efficient, accurate, and scalable for next-generation wireless networks
